# Lead Capture Automation (n8n)

This project contains a simple, reliable lead capture automation.

## What this workflow does
- Captures lead submissions from Google Forms
- Logs lead data in Google Sheets
- Sends confirmation email to the lead
- Notifies the coach immediately
- Stops

## What it does NOT do
- No lead scoring
- No prioritization
- No follow-ups
- No reminders

## Tools Used
- Google Forms
- Google Sheets
- Gmail
- n8n

## Mental Model
This automation behaves like a receptionist:
it receives the inquiry, logs it, confirms receipt, notifies the coach, and goes silent.
